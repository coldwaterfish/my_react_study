### 运算

#### 运算符

##### 取余

```js
// %
let a = 54;
console.log(a % 10);// 4
```

##### 次方

```js
// **
console.log(2 ** 3);// 8

Math.pow(基数,指数)
console.log(Math.pow(2, 3));
```

##### 开方

```js
// 开方
Math.sqrt(9);//3
```

##### 位运算符

```js
// ^:异或运算
1^3结果是2
（1）因为：1的二进制表达为“0001”。
（2）而3的二进制表达为“0011”。
（3）当1^3即运用异或运算，运算后的结果是“0010”，转换为十进制之后，即为“2”
```
