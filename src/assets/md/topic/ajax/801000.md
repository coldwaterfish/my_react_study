### http请求

#### get和post的区别

```js
// 参数位置和安全性
get请求的参数包含在url中，post请求的参数放在请求体中
post请求安全性大于get请求
```

```js
// 数据大小
http协议规范没有数据大小限制
各浏览器对url的长度有限制2k-8k之间
url长度过长返回414错误
```

```js
// 请求次数
Post请求发送一次还是两次，取决与浏览器自己的实现，
比如上传文件的场景，请求头中有文件的名字等信息，
请求体是文件本身，如果文件很大就可以分别发送，
先验证请求头，文件名等信息是否符合要求，
如果不符合就没有必要发送请求体了。
另外Ajax是会分两次发送的，这是因为Ajax使用了XMLHttpRequest API，
浏览器会先发送请求头在发送请求体，但是火狐浏览器不会。
所以Post请求到底是发送一次还是两次，取决于浏览器的实现，不论一次还是两次都是符合标准的
```
